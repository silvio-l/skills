# Prompt Library

Copy-paste starting points for the Claude Design studio, derived from the workflows in `studio-workflow.md` and `motion-graphics.md`. Fill in the bracketed placeholders. Kept in their original language where that's how they were phrased.

## Design system briefing

```
Our company is [name] — [one-sentence product description]. It solves
[the core problem] for [target audience]. The tone should be
[e.g. clean, analytical, data-forward / warm, playful, approachable].
We do [/ do not] have an existing logo. Copy language: [English/German/...].
This design system should cover: [web app / marketing website / both],
with [standard set of buttons, inputs, etc. / extended component depth].
```

## Multi-page app prototype

```
Kannst du mir bitte einen App-Prototyp mit mehreren Seiten erstellen,
basierend auf dem Designsystem [Name]? Der Kern-Workflow ist:
[Schritt 1] → [Schritt 2] → [Schritt 3]. [Zusätzliche Anforderungen,
z. B. welche externen Systeme angebunden werden, wie interaktiv der
Prototyp sein soll].
```

## Pitch deck

```
Es steht ein Investorenpitch bevor für [Runden-Typ, z. B. Pre-Seed].
Ich benötige ein Pitchdeck für [Produktname], [Anzahl] Folien.
Kernkunden sind [Zielgruppe]. Wir wollen [Betrag] raisen, aktuelle
Traction: [Zahlen]. Analysiere bitte auch den Wettbewerb. Team:
[Namen/Rollen]. Differenzierung: [Kernpunkte]. Tonalität:
[z. B. nüchtern, datengetrieben]. Sprache: [Deutsch/Englisch].
```

## Landing page

```
Kannst du bitte eine Landing Page für [Produktname] erstellen, basierend
auf dem Designsystem [Name]? Primäre Zielgruppe: [wer]. Haupt-CTA:
[z. B. Warteliste / Kauf]. Struktur: Hero mit Produktmockup, Feature
Grid, [weitere Sektionen wie Testimonials/FAQ]. Pricing: [Betrag/Plan].
Erstelle [Anzahl] Hero-Varianten. Sprache: [Deutsch/Englisch].
```

## Infographic / one-pager

```
Please create a one-page infographic for [product/company], based on
the [design system name] design system. It should cover: [key points
to visualize — problem, solution, key stats, etc.].
```

## Wireframe

```
I need a wireframe for the [product/company] website/app, [N] pages:
[list the pages]. Keep it unstyled — boxes and labels only, no color —
so we can validate the layout before any visual design work starts.
```

## Motion graphics — start/end screenshots

```
I'm attaching two screenshots of a motion graphic I want to recreate —
the start frame and the end frame. [Describe the animation step by
step: what happens, in what order, and how each element transitions.]
I want to adapt it for [your content instead of the original's].
```

## Motion graphics — transcript-driven

```
Here is a timestamped transcript (SRT-style) of my video/presentation.
Create chapter/section motion graphics synced to these timestamps:
[paste transcript]. Use this screenshot for on-brand styling: [attach].
```

## Motion graphics — animate any UI

```
Create a video of [product] being used. It should feel like a screen
recording of a real user. Animation 1: [zoom into X]. Animation 2:
[type Y]. Animation 3: [result appears]. [Attach the screenshot and the
captured element code.]
```
