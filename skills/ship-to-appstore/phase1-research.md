# Phase 1 — Freshness Research

> **Stub — implemented in slice 02.**

This phase performs web research to retrieve the current Apple requirements before
providing any release-step guidance. Nothing from training memory is used for
step instructions without a freshness check.

Topics to search for (with the current year/month in the query):
- Minimum Xcode version accepted by App Store Connect
- Minimum iOS deployment target required
- Required screenshot sizes (iPhone 6.9", 6.5", iPad Pro, etc.)
- Privacy nutrition label current requirements
- Account deletion requirement status
- Export Compliance / encryption declaration rules
- App Review guideline changes
- Any ASC API changes relevant to Phase 2
